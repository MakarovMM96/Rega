# Инструкция по установке и запуску приложения

## Удаление AI-функций

В этом репозитории были удалены все AI-функции для обеспечения стабильной работы приложения без необходимости в API-ключах. Вот что было изменено:

1. Удален файл `/services/geminiService.ts`
2. Из компонента `App.tsx` удален импорт функции `generateHypeMessage`
3. Из `package.json` удалена зависимость `@google/genai`
4. Обновлена логика в `handleSubmit`, чтобы не использовать AI-функции
5. Обновлен `README.md`, чтобы отразить изменения

## Установка зависимостей

Для установки зависимостей выполните:

```bash
npm install
```

Если возникают проблемы с памятью при установке, попробуйте:

```bash
npm install --no-audit --no-fund --prefer-offline
```

Или используйте альтернативный способ:

```bash
# Удалите node_modules и package-lock.json, если они есть
rm -rf node_modules package-lock.json
npm install
```

## Запуск приложения

После установки зависимостей запустите приложение:

```bash
npm run dev
```

## Функциональность

Приложение теперь полностью функционирует без AI-сервисов:
- Форма регистрации работает корректно
- Данные сохраняются через сервис yandexService
- Вместо AI-сообщения используется подготовленное сообщение с данными пользователя
